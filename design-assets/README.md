## Workflow

1. **Capture Current Screens**: Take screenshots of all modules and save in `current-screens/`
2. **Use Stitch AI**: Use prompts from `STITCH_PROMPTS.md` to generate designs
3. **Export Designs**: Save exported HTML/CSS or Figma files in `stitch-exports/`
4. **Document Design System**: Create design system documentation in `design-system/`
